Detailed Description

Instance Configuration

AVS_Instance_Factory allows to configure the AVS instance using several parameters. A zero in a field, means that the engine will use a default parameter. After the call to AVS_CreateInstance , all fields with 0 will be updated with a default value used by the engine. Notice, some fields are mandatory to connect the server (client Id, client Secret, product Id, serial number)
